Media sourcing policy

This website uses only public-domain media. Each visual asset should be logged in media-log.json with source URL and license status.

Approved source examples

  • Wikimedia Commons

    Use files explicitly marked Public Domain or CC0.

  • The Met Open Access

    Use CC0 collection assets for illustrations and textures.

  • Rijksmuseum Open Data

    Use public-domain artworks for thematic details.

  • Finnish cultural archives

    Verify rights per item before use even when source is institutional.

Maintenance checklist

  • Before publishing new image

    Confirm public-domain status and save source URL.

  • After adding image file

    Add a record into media-log.json with date and notes.

  • Quarterly review

    Check links and remove assets with uncertain status.
